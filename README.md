# Galarraga Research - Landing Page de Investigación Médica

Una landing page moderna y profesional para acceder a investigaciones médicas confiables.

## 🏥 Características

- **Diseño Profesional**: Tema médico con colores azules profesionales
- **Buscador Simulado**: Demo funcional de búsqueda de artículos médicos
- **Responsive Design**: Adaptado para móviles, tablets y desktop
- **Accesibilidad**: Navegación por teclado y screen readers
- **SEO Optimizado**: Meta tags y estructura semántica

## 🚀 Instalación y Uso

### Requisitos
- Navegador web moderno
- Node.js (opcional, para servidor de desarrollo)

### Instalación
1. Clona o descarga el proyecto
2. Abre `index.html` en tu navegador
3. ¡Listo! El sitio está funcionando

### Para desarrollo local
```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm start

# Servidor de producción
npm run serve
```

## 🎨 Secciones de la Landing Page

1. **Header**: Logo y navegación mínima
2. **Hero**: Título principal y CTAs
3. **Beneficios**: 3 tarjetas con ventajas clave
4. **Demo**: Buscador simulado funcional
5. **Testimonios**: Opiniones de profesionales
6. **CTA Final**: Suscripción de email
7. **Contacto**: Información de contacto
8. **Footer**: Enlaces y redes sociales

## 🔧 Funcionalidades

- **Búsqueda Simulada**: Funciona con sugerencias y resultados
- **Suscripción de Email**: Formulario funcional
- **Navegación Suave**: Scroll entre secciones
- **Menú Móvil**: Hamburguesa responsive
- **Efectos de Scroll**: Header dinámico

## 📱 Responsive Design

- **Mobile First**: Diseño optimizado para móviles
- **Breakpoints**: sm (640px), md (768px), lg (1024px)
- **Grid Adaptativo**: Se ajusta automáticamente
- **Componentes Flexibles**: Botones y formularios adaptativos

## 🎯 Optimizaciones

- **CSS Modular**: ITCSS con componentes separados
- **JavaScript Vanilla**: Sin dependencias externas
- **Imágenes Optimizadas**: Lazy loading
- **Código Limpio**: Fácil mantenimiento
